C Call Hierarchy Explorer 1.1.12
================================

Windows 10/11 64-bit용 C 함수 호출 계층 분석 프로그램입니다.

설치 방법
---------
1. "C-Call-Hierarchy-Explorer-Setup-1.1.12.exe"를 실행합니다.
2. 사용자 계정의 LocalAppData\Programs 폴더에 설치됩니다.
3. 바탕 화면과 시작 메뉴에 바로가기가 생성됩니다.

포터블 실행
-----------
설치하지 않으려면 "C-Call-Hierarchy-Explorer-Portable-1.1.12.exe"를 직접 실행할 수 있습니다.

포함 구성요소
-------------
- PySide6
- Tree-sitter / tree-sitter-c
- libclang

Python이나 위 라이브러리를 별도로 설치할 필요가 없습니다.

제거 방법
---------
Windows 설정 > 앱 > 설치된 앱에서 "C Call Hierarchy Explorer"를 제거합니다.

참고
----
개인 서명 인증서가 적용되지 않은 설치 파일이므로 Windows SmartScreen 확인이 표시될 수 있습니다.
