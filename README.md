# C Call Hierarchy Explorer

Windows용 C 언어 함수 호출 계층 분석 프로그램입니다.

## 주요 기능

- `.c`, `.h` 함수 분석
- main 및 ISR 호출 계층 표시
- Tree-sitter 및 libclang 기반 분석
- 2초 단위 파일 변경 감시
- Excel 호출 트리 출력
- 프로젝트 분석 상태 캐시
- GitHub Releases 자동 업데이트

## 다운로드

최신 설치 파일은 오른쪽의 **Releases**에서 받을 수 있습니다.

- Setup: 일반 설치 버전
- Portable: 설치하지 않고 실행하는 버전

## 시스템 요구사항

- Windows 10/11 64-bit
- 별도 Python 설치 불필요

## 주의사항

이 저장소는 프로그램 배포 전용 저장소입니다.
